# Yume Project ｜肖笑笑的海市超高楼人生

一个原创漫画项目，由 HTML + CSS 驱动，记录外包女孩肖笑笑在666层科技公司“货梯+树景房”的暴笑生活。

📘 **在线阅读地址**
